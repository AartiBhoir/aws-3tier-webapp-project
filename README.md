AWS 3-Tier Web Application Project

 Project Overview

This project demonstrates a production-style 3-tier architecture deployed on AWS using Amazon EC2 and Amazon RDS.

AWS Services Used

* Amazon EC2
* Amazon RDS
* VPC
* Security Groups
* Amazon Linux

 Architecture

User → EC2 Web Server → RDS MySQL Database

 Features

* Linux web server deployment
* Apache web server setup
* MySQL database integration
* EC2 to RDS connectivity
* Database table creation and data insertion

 Commands Used

Stored inside commands.txt

 Database Queries

Stored inside database.sql

 Project Outcome

Successfully connected EC2 instance with RDS database and performed database operations using MySQL client.
